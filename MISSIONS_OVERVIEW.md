## Mission Difficulty Progression

The campaign is designed to build excitement:
- **Weeks 1-2**: Introduction, simple missions (100-200 points)
- **Weeks 3-4**: Mid-tournament engagement (100-250 points)
- **Weeks 5-6**: Finals excitement, higher rewards (100-300 points)

## Setup Instructions

1. **Open Supabase Dashboard** → SQL Editor
2. **Copy contents** of `supabase/schema.sql`
3. **Paste and Run** the SQL
4. **Verify** all 42 missions are created in the `missions` table
5. **Activate weeks** as needed by setting `is_active = true`

All missions are currently set to `is_active = true` for demonstration purposes. In production, you would activate each week's missions at the appropriate time.
